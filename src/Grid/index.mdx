# Grid(栅格)

对 antd Grid 的二次封装,主要支持 Row 传入 ColProps 传入 Col 的 context

## 属性(Properties)

参考[antd grid](https://ant.design/components/grid-cn/)

### Row

<Props of={Row} />

Row 比 antd Row 增加了一个可选参数 colProps 即 antd Col 的所有属性,会传给 Col 然是可以被 Col 的属性覆盖

### Col

<Props of={Col} />

## 基本用法(Basic usage)

<Playground>
  <Row colProps={{ span: 8 }}>
    <Col>1111111111</Col>
    <Col>1111111111</Col>
    <Col span={24}>1111111111</Col>
    <Col>1111111111</Col>
  </Row>
</Playground>
